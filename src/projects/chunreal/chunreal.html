Chunreal: strongly-timed audio programming in Unreal Engine 5 (2023)

Chunreal is a real-time audio synthesis plugin for Unreal Engine 5 (UE5) on Windows and macOS. The plugin utilizes the existing digital signal processing (DSP) graph in UE5 called MetaSounds. Chunrealâ€™s unique MetaSound node embeds ChucK - a strongly-timed music programming language, and allows users to compile code on-the-fly for audio synthesis and analysis.

The following GitHub repository contains the template Unreal Engine project and step-by-step tutorials: https://github.com/ccrma/chunreal
