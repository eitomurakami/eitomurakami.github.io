Max/MSP DJ Player & Mixer (2020)

A 2-Channel Virtual DJ System using Max/MSP. Some of the main features include headphone cue monitoring, a memory cue, and a 3-band equalizer for each player.
The following video is a DJ performance using this patch and Roland TR-8S as a MIDI controller.
